Online Invoicing Laboratory



This is my Java final project for Software Development Academy exam, with the theme of online invoicin for companies, from different countries.

It is developed in IntelliJ IDEA from Jetbreans, using Java programming language JDK1.8, Maven and JPA repositories, Spring MVC and SpringBoot frameworks for back-end, and HTML, CSS, JavaScript programming language, Thymeleaf, Bootsrtrap for front-end. I also used Jaspersoft for creating PDF and HTML reports.
The database is written in SQL Workbench.
The invoices are generated by a company that sells the on-line app to its companies'clients.


The project have different tasks, such as:

  • Management of users with different roles;
  • Add, edit or delete a company (a user);
  • Add, edit or delete invoices and payments by the users;
  • Add, edit or delete clients for each user;
  • Add, edit or delete products and services provided by the user;
  • Generate reports and statistics for the user (company) for issued invoices, products/services sold, payments in total invoices or for each client, most sold products or services etc.

The project will have different tables with different relationship among them, such as:

  1. Users;
  2. Clients;
  3. Products/services;
  4. Invoices;
  5. Lines of invoices;
  6. Payments
  7. Authorities

    8. and static tabels, such as:

  8. City;
  9. Country;
  10. User type;
  11. Product type;
  12. Unit type;
  13. Payment type;

Note: The list of relational tables and entities is not final, because it can be modified for the users' purposes.

The Maven project will use JPA repositories, HTML templates and CSS styles and will provide an on line template for issuing invoices by the signed in users, to their clients, along with payments and providing reports for analysis of the past activities in order to take better decisions in the future and see the market trends.
